Release 1.9.1.0

  1. NEW: The launcher icon was made smaller vertically, and the feature graphic is updated. Thanks Rinke-Jan Schaper!
  2. NEW: 'Grid mode' toggle replaced with 'Grid size' option. Will save different values in portrait/landscape.
  3. NEW: Local folders will automatically refresh when coming back to the app, e.g. when you go to the text editor and come back.
  4. CHANGE: The 'Icon Color' setting is now called 'Thumbnail Color', it allows you to select more colors than just the default/primary/accent.
  5. IMPROVE: Small speed improvements to root access. Many more improvements are planned.
  6. FIX: You can no longer use dark colors for accent colors, this caused a lot of visual issues for some.
  7. FIX: Renaming will no longer append the old name to the end of the new name.
  8. FIX: Some setting descriptions were cut off with certain languages.
  9. FIX: Both hidden folders and files are semi-transparent now.
  10. FIX: An enormous amount of crash fixes based on automatic crash reports.
  11. FIX: -1 was shown as directory size in root folders before.
  12. TRANSLATION: Lots of translation updates.
